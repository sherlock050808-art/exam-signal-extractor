# 开卷考试策略

## 核心原则

**同一套七层框架，更高的覆盖阈值。**

开卷考试允许带资料进场，因此不需要把所有内容背下来，但七层地图中的 🟢🟡🟠 层都要进入资料。差异在于：

- 🟢 必考层：完整答案，能直接照抄
- 🟡 高概率层：完整答案，能直接照抄
- 🟠 可能层：至少覆盖名词解释或简答采分点，便于临场查找
- 🔴 排除层：明确标记，不写入资料
- ⚡ 陷阱区：整理易混点和区分方式
- 🔗 融合区：准备完整综合分析答案

## 与闭卷的关键差异

| 维度 | 开卷 | 闭卷 |
|------|------|------|
| 框架 | 同一七层地图 | 同一七层地图 |
| 覆盖阈值 | 高（🟠 也要覆盖） | 相对低（🟠 了解即可） |
| 资料厚度 | 厚，但必须可检索 | 薄，按层清晰 |
| 记忆强度 | 识别 + 定位 | 默写 + 调用 |
| 索引 | 必须有章节/题型/关键词索引 | 可按概率分层 |

## 操作要点

### 1. 课件标题扫描（最重要）

用 PDF 文本提取工具，把课件中所有标题级、加粗、独立条目的概念全部列出来。

每个概念至少写一句话定义，形成名词解释清单；对老师强调的概念则要写成完整简答。

### 2. 七层分别处理

| 层次 | 处理方式 |
|------|---------|
| 🟢 必考层 | 完整答案 + 放在资料最前面或单独成节 |
| 🟡 高概率层 | 完整答案 + 做好索引 |
| 🟠 可能层 | 名词解释 / 简答采分点，便于临场扩展 |
| 🔴 排除层 | 明确写“不考”，不占用资料篇幅 |
| ⚡ 陷阱区 | 对比表形式整理 |
| 🔗 融合区 | 准备 1-2 个完整综合分析答案 |

### 3. 资料必须可检索

- 按章节分页/分节
- 每个题型单独一块
- 增加关键词首字母索引
- 使用颜色标记重要性

### 4. 完整答案优先

简答题和综合分析题不要只写提纲，要写可以直接抄的完整段落。

### 5. 考前检查

- 资料是否打印成功？
- 中文显示是否正常？
- 页码是否连续？
- 索引是否完整？
